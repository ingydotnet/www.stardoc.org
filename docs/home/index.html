Stardoc is an Acmeist documentation framework. It allows you to write software in one of the following input formats:

  • POD
  • ReSTructured Text
  • Markdown
  • MediaWiki
  • Socialtext
  • HTML
  • DocBook

Stardoc can convert any of those input formats to these output formats:

  • POD
  • ReSTructured Text
  • Markdown
  • MediaWiki
  • Socialtext
  • HTML
  • DocBook
